About Me

 게임, 방산, XR, 디지털 트윈 등 다양한 산업 분야에서 클라이언트 개발 경험을 쌓으며, 문제 해결 중심의 개발 역량을 키워왔습니다. Unity, Unreal Engine, 자 체 게임 엔진 등 다양한 렌더링 엔진 환경부터, Linux 임베디드 시스템 프로그래밍까지 다양한 영역의 개발을 수행해왔습니다.

 게임 개발사 인턴으로 AAA급 신작 프로젝트에 참여해 UI 기능을 구현했습니다. 자체 엔진 UI 구조를 파악하고 엔진 처리 흐름에 맞춰 기능을 구현했으며, 캐릭터 전환을 위한 원형 퀵슬롯 UI와 월드맵 오브젝트 밀도 시각화 UI를 개발했습니다. 구현한 월드맵 UI는 디버깅에 도움이 되었다는 피드백을 개발 PD에게 받았습니다.
 항공소프트웨어 분야에서는 HUMS 개발을 담당했습니다. 기존에 2만 줄 이상의 코드가 단일 main 파일에 작성된 시스템을 객체지향적으로 처음부터 재설계했 습니다. 데이터 처리 과정을 데이터 수신정제저장의 3단계로 멀티스레드 파이프라인으로 나누어 구현했습니다. 또한 UDP 통신 중 하드웨어 문제로 발생한 데 이터 유실 문제를 타임스탬프 기반으로 감지하고, 누락된 데이터를 자동으로 재요청하는 방식으로 해결해 시스템의 신뢰성을 확보했습니다. 그 결과, 전 장비가 수락시험에 통과하고 납품 일정을 준수할 수 있었습니다.
 XR 및 디지털트윈 분야에서도 여러 실무 프로젝트에 참여했습니다. HoloLens 기반 XR 프로젝트에서는 기존 클래스를 리팩토링해 유지보수성을 높이고, Unity Job System을 적용해 모델 로딩 병목을 개선했습니다. 디지털 트윈 기반 교차로 시뮬레이션 및 관제 프로젝트에서는 CCTV 영상을 통해 차량 위치를 인식하 고, 이를 3D 가상 공간에 실시간 매핑하는 기능을 구현했습니다. 모바일 AR 기반 화재 진압 시뮬레이터에서는 Unreal Engine 5를 기반으로 클라이언트를 개발 하고, 실제 화재 대응 절차에 따라 동작하는 시나리오 구현을 담당했습니다.

 다양한 환경에서수행한 프로젝트 경험을 바탕으로, 새로운 방식의 접근과 아이디어로 문제 해결하려 노력하고 있습니다.
